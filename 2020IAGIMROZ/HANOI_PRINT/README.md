Twoim zadaniem jest przełożenie wieży Hanoi z pierwszego słupka na trzeci słupek.

Wejście
Jedna liczba naturalna n(1 <= n <= 20).

Wyjście

Przykład

3

1 3
1 2
3 2
1 3
2 1
2 3
