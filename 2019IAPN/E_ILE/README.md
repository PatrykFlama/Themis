# Liczba dzielników
###### Problem code: E_ILE \| Time: 1.00 s \| Memory: 16 MB \| Solved: yes \| print

Oblicz ile dzielników ma liczba n (wliczając w nie 1 oraz n).

#### Wejście
Jedna liczba całkowita n (2<=n<=10^9).

#### Wyjście
Liczba dzielników n.

#### Przykład
Dla danych wejściowych

```
6
```
poprawną odpowiedzią jest
```
4
```
