# Suma ciągu liczb
###### Problem code: RAM01 \| Time: 1.00 s \| Memory: 8 MB \| Solved: yes \| print

Zadanie
Oblicz sumę ciągu liczb zakończonego liczbą 0.

#### Wejście
a1 a2 a3 ... an 0

#### Wyjście
a1 + a2 + a3 + ... + an

#### Przykład
Dla danych wejściowych

```
1 2 3 0
```
poprawną odpowiedzią jest
```
6
```
