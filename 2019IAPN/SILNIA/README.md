# Silnia
###### Problem code: SILNIA \| Time: 1.00 s \| Memory: 32 MB \| Solved: yes \| print

Policz n!.

#### Wejście
W pierwszej i jedynym wierszu wejścia jest jedna liczba n (0≤n≤10).

#### Wyjście
Wypisz jedną liczbę równą n!.

#### Przykład
Dla danych wejściowych

```
4
```
poprawną odpowiedzią jest

```
24
```
